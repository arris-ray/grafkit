```
A toolkit of Grafana utilities.

Version: 0.1.0-alpha

Usage:
  grafkit <subcommand> [--subcommand-options] [<arguments>]
  grafkit -h | --help
  grafkit --version

Options:
  -h --help  Display this help information.
  --version  Display version information.

Help:
  grafkit help [<subcommand>]

Available subcommands:
  build
  replace
  shell
  build
  example
  help
  shell
  subcommands
  version
```
